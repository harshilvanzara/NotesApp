# 📝 Notes App (Java File I/O)

## 📌 Description
A simple text-based Notes Manager built using Java.
It allows users to write and read notes using file handling.

## 🚀 Features
- Write notes to file
- Read saved notes
- Simple menu-driven program

## 🛠️ Technologies Used
- Java
- FileWriter
- FileReader
- BufferedReader

## ▶️ How to Run
1. Compile the program:
   javac NotesApp.java

2. Run the program:
   java NotesApp

## 📂 Output
- Notes are stored in `notes.txt`
